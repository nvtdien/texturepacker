Help

If you are having problems with any of the textures, use MC Patcher. It should iron out most problems with dodgy textures.

Installation Instructions:

1. Download "Olive.zip".
2. Press "Windows + R".
3. Navigate to ".minecraft".
4. Find "texturepacks".
5. Move "Olive.zip" into the "texturepacks" folder.
6. Run the game.
7. Select "Olive" from the texture packs menu.
8. Play and enjoy!
