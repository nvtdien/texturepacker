Minecraft Texture Maker by pdc

Define recipes that specify how to create a texture packs for Minecraft from a collection of PNG files in a directory or from resources pulled out of existing texture packs.

Use maketexture to build the texture pack automatically—no Photoshop required.

Status

Until I create a proper installer this is a technology preview for Python programmers. I intend to create an installer compatible with some subset of EasyInstall, Distribute, Pip, and the various other heirs and successors to Distutils, but until then you can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/pdc/minecraft-texture-maker
get the source code on GitHub : pdc/minecraft-texture-maker
