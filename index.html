Minecraft Texture Maker

Build & remix texture packs for Minecraft

Define recipes that specify how to create a texture pack for Minecraft from PNG files, or from resources pulled out of existing texture packs. Use maketexture to build the texture pack automaticallyâ€”no Photoshop required.

Dependencies

Requires Python 2.7 and several Python modules listed in the file REQUIREMENTS.

Install

For the moment this is what you might call a technology preview: you can download the code and run the scripts, but you need to know your way around a Python program.

You can also clone the project with Git by running: 

$ git clone git://github.com/pdc/minecraft-texture-maker

Then (possibly in a virtual environment created with virtualenv), install the dependencies: 

$ pip install -r REQUIREMENTS

You can then run tests and build the sample texture packs: 

$ nosetests tests
$ script/maketexture -v texturepacks/groovystipple.json

I intend to add proper installer files soon. The work backlog is in the wiki for now.
